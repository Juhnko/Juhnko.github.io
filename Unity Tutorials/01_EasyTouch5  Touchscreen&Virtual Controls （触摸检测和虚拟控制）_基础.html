01、One Finger

  • This example shows you the different gesture recognition capabilities for EasyTouch with 1 finger.  这个例子向你展示了EasyTouch用一根手指的不同手势识别功能。
  • EasyTouch fires events depending on the actions that occur. A gesture can be decomposed into several events.  EasyTouch根据发生的动作触发事件。一个手势可以分解成几个事件。
  • All events are sent with a class as parameter that will allow you to know all datas on the current gesture.  所有事件发送时都带有一个类作为参数，它将允许你知道当前手势的所有数据。
  • Two methods are available to intercept these events: 有两种方法可以拦截这些事件:
    • Programmatically 编程的方式
      • By subscribing to the events you want.  订阅你想要的事件
    • EasyTouch_Trigger 触发器
      • Add this component to a gameobject, it allows you to call existing methods relative to the events you want to receive, without write a line of code. 将这个组件添加到游戏对象中，它允许你调用与你想要接收的事件相关的现有方法，而不需要编写一行代码。

using System.Collections;
using HedgehogTeam.EasyTouch;

private Color startColor;

EasyTouch.On_TouchDown += On_TouchDown;
EasyTouch.On_TouchUp += On_TouchUp;

}
}
// During the touch is down

private void On_TouchUp(Gesture gesture){

}
}

// Verification that the action on the object
if (gesture.pickedObject == gameObject){

Vector3 scale = transform.localScale ;
transform.localScale = new Vector3( scale.x - zoom, scale.y -zoom, scale.z-zoom);
textMesh.text = "Delta pinch : " + gesture.deltaPinch.ToString();
}
}

private void On_PinchOut(Gesture gesture){
// Verification that the action on the object
if (gesture.pickedObject == gameObject){

Vector3 scale = transform.localScale ;
transform.localScale = new Vector3( scale.x + zoom, scale.y +zoom,scale.z+zoom);
textMesh.text = "Delta pinch : " + gesture.deltaPinch.ToString();
}
}
// At the pinch end
private void On_PinchEnd(Gesture gesture){
if (gesture.pickedObject == gameObject){
transform.localScale =new Vector3(2.5f,2.5f,2.5f);

01、AutoSelect

  
      EasyTouch allows automatic 3D object selection.   EasyTouch允许自动选择3D对象。
      The class passed as a parameter to the events,  has a property that allows you to know this object.  这个类可以被传递，作为一个参数传递给事件，有一个属性允许您知道这个对象。
      Also you can choose the layers on which you want an automatic selection.  你也可以选择你想要自动选择的图层。

      By default the automatic selection is done at the touch start.  默认情况下，自动选择是在触摸开始时完成的。
      The selected object will be sent as a parameter with all events, up to the release of the touch.  被选择的对象将作为参数发送所有事件，直到触摸的释放。
      You can choose to update in real time the object under the touch.  你可以选择实时更新触摸下的对象。
      You can also know the Current object via a method if auto update is not active.  如果自动更新没有激活，你也可以通过一个方法知道当前对象。
using UnityEngine;
using UnityEngine.UI;
using System.Collections;
using HedgehogTeam.EasyTouch;
public class MultiLayerTouch : MonoBehaviour {
public Text label;
public Text label2;
void OnEnable(){
EasyTouch.On_TouchDown += On_TouchDown;
EasyTouch.On_TouchUp += On_TouchUp;
}
void OnDestroy(){
EasyTouch.On_TouchDown -= On_TouchDown;
EasyTouch.On_TouchUp -= On_TouchUp;
}
void On_TouchDown (Gesture gesture)
{
if (gesture.pickedObject!=null){
if (!EasyTouch.GetAutoUpdatePickedObject()){
label.text = "Picked object from event : " + gesture.pickedObject.name + " : " + gesture.position;
}
else{
label.text = "Picked object from event : " + gesture.pickedObject.name + " : " + gesture.position;
}
}
else{
if (!EasyTouch.GetAutoUpdatePickedObject()){
label.text = "Picked object from event : none";
}
else{
label.text = "Picked object from event : none";
}
}
label2.text = "";
if (!EasyTouch.GetAutoUpdatePickedObject()){
GameObject tmp = gesture.GetCurrentPickedObject();
if (tmp != null){
label2.text = "Picked object from GetCurrentPickedObject : " + tmp.name ;
}
else{
label2.text = "Picked object from GetCurrentPickedObject : none";
}
}
}
void On_TouchUp (Gesture gesture)
{
label.text="";
label2.text="";
}
}

using UnityEngine;

public class UIWindow : MonoBehaviour, IDragHandler, IPointerDownHandler{

}

}
}
using UnityEngine;

{

{
drag = false;
if (gesture.isOverGui){
if (gesture.pickedUIElement == gameObject || gesture.pickedUIElement.transform.IsChildOf( transform)){
transform.SetAsLastSibling();
drag = true;
}
}
}

{

if ((gesture.pickedUIElement == gameObject || gesture.pickedUIElement.transform.IsChildOf( transform)) && drag){

}
}

The new Unity UI doesn't support pinch & twist gestures on its elemenst. 新的Unity UI在其元素上不支持缩放和扭曲手势。
You can use EasyTouch to overcome this deficiency. 你可以使用EasyTouch来克服这个缺陷。

The Unity UI compatibility should be disabled in order to receive events from EasyTouch when the touch is over a UI element.
Unity的UI兼容性应该被禁用，以便在触摸在UI元素上时从EasyTouch接收事件。
using UnityEngine;
using UnityEngine.EventSystems;
using HedgehogTeam.EasyTouch;
public class UITwist : MonoBehaviour{
public void OnEnable(){
EasyTouch.On_Twist += On_Twist;
}
public void OnDestroy(){
EasyTouch.On_Twist -= On_Twist;
}
void On_Twist (Gesture gesture){
if (gesture.isOverGui){
if (gesture.pickedUIElement == gameObject || gesture.pickedUIElement.transform.IsChildOf( transform)){
transform.Rotate( new Vector3(0,0,gesture.twistAngle));
}
}
}
}
using UnityEngine;
using System.Collections;
using HedgehogTeam.EasyTouch;
public class UIPinch : MonoBehaviour {
public void OnEnable(){
EasyTouch.On_Pinch += On_Pinch;
}
public void OnDestroy(){
EasyTouch.On_Pinch -= On_Pinch;
}
void On_Pinch (Gesture gesture){
if (gesture.isOverGui){
if (gesture.pickedUIElement == gameObject || gesture.pickedUIElement.transform.IsChildOf( transform)){
transform.localScale = new Vector3(transform.localScale.x + gesture.deltaPinch * Time.deltaTime, transform.localScale.y+gesture.deltaPinch * Time.deltaTime, transform.localScale.z );
}

}
}
using UnityEngine;
using System.Collections;
using HedgehogTeam.EasyTouch;
public class UIDrag : MonoBehaviour {
private int fingerId = -1;
private bool drag = true;
void OnEnable(){
EasyTouch.On_TouchDown += On_TouchDown;
EasyTouch.On_TouchStart += On_TouchStart;
EasyTouch.On_TouchUp += On_TouchUp;
EasyTouch.On_TouchStart2Fingers += On_TouchStart2Fingers;
EasyTouch.On_TouchDown2Fingers += On_TouchDown2Fingers;
EasyTouch.On_TouchUp2Fingers += On_TouchUp2Fingers;
}
void OnDestroy(){
EasyTouch.On_TouchDown -= On_TouchDown;
EasyTouch.On_TouchStart -= On_TouchStart;
EasyTouch.On_TouchUp -= On_TouchUp;
EasyTouch.On_TouchStart2Fingers -= On_TouchStart2Fingers;
EasyTouch.On_TouchDown2Fingers -= On_TouchDown2Fingers;
EasyTouch.On_TouchUp2Fingers -= On_TouchUp2Fingers;
}
void On_TouchStart (Gesture gesture){
if (gesture.isOverGui && drag){
if ((gesture.pickedUIElement == gameObject || gesture.pickedUIElement.transform.IsChildOf( transform)) && fingerId==-1){
fingerId = gesture.fingerIndex;
transform.SetAsLastSibling();
}
}
}
void On_TouchDown (Gesture gesture){
if (fingerId == gesture.fingerIndex && drag){
if (gesture.isOverGui ){
if ((gesture.pickedUIElement == gameObject || gesture.pickedUIElement.transform.IsChildOf( transform)) ){
transform.position += (Vector3)gesture.deltaPosition;
}
}
}
}
void On_TouchUp (Gesture gesture){
if (fingerId == gesture.fingerIndex){
fingerId = -1;
}
}
void On_TouchStart2Fingers (Gesture gesture)
{
if (gesture.isOverGui && drag){
if ((gesture.pickedUIElement == gameObject || gesture.pickedUIElement.transform.IsChildOf( transform)) && fingerId==-1){
transform.SetAsLastSibling();
}
}
}
void On_TouchDown2Fingers (Gesture gesture)
{
if (gesture.isOverGui){
if (gesture.pickedUIElement == gameObject || gesture.pickedUIElement.transform.IsChildOf( transform)){
if ((gesture.pickedUIElement == gameObject || gesture.pickedUIElement.transform.IsChildOf( transform)) ){
transform.position += (Vector3)gesture.deltaPosition;
}
drag = false;
}
}
}
void On_TouchUp2Fingers (Gesture gesture)
{
if (gesture.isOverGui){
if (gesture.pickedUIElement == gameObject || gesture.pickedUIElement.transform.IsChildOf( transform)){
drag = true;
}
}
}
}

This example shows some of the possibilities of QuickGestures Components.

They allow you to easily capture the gesture performed on the objects , you just have to call a script to perform an action. Or perform simple actions such as : translate, rotate, scale.

These components are configurable for 1 finger or 2 fingers gesture .

它们将自动检测游戏对象的类型(3D, Sprite, UI)，并根据其性质配置EasyTouch。
